# 会议纪要生成模板

## 纪要结构

```markdown
# 会议纪要

**时间**：YYYY-MM-DD
**参与人**：姓名1、姓名2、...

---

## 一、主题标题

要点内容...

---

## 二、下一个主题

...

---

## 后续任务

| 负责人 | 任务 | 优先级 |
|--------|------|--------|
| ... | ... | ... |

---

*纪要生成时间：YYYY-MM-DD*
*转录工具：FunASR (SenseVoiceSmall + fsmn-vad + CAM++)*
```

## 写作要求

1. **用真实姓名**，不要出现 SPEAKER_XX
2. **提炼要点**，不要逐字照搬转录内容
3. **保留关键结论**、决策、技术细节
4. **修正错别字**，ASR 对同音字可能误识别
5. **结构化呈现**，用表格、列表、代码块增强可读性
6. **中文输出**

## 常见纪要板块

根据会议内容，可能包含以下板块：

- **核心要求/指示**：领导提出的工作要求
- **工作汇报**：每人本周完成事项（表格形式）
- **技术问题结论**：问题描述 + 解决方案
- **策略讨论**：讨论要点 + 结论
- **技术讲解**：架构、方案、流程图
- **后续任务**：负责人 + 任务 + 优先级
